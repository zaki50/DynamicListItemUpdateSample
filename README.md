リストアイテムをタップするとダイアログが表示され、ダイアログでの選択結果に応じで対応するリストアイテムが更新されます。

    ./gradlew clean assemble

でビルドできます。正常にビルドが完了すると、 app/build/outputs/apk/app-debug.apk が生成されます。
